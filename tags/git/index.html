git命令大全

指令 作用 git init 创建版本库 git add 将工作区文件放到暂存区,可反复多次使用，添加多个文件 git commit 将暂存区文件放到分支 git statu 查看仓库当前状况（有多少文件修改，在什么位置） git diff 查看文件具体做了什么修改 git log 查看提交历史，以便确定要回退到哪个版本 git reset 重新设定版本
